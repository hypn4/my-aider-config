# my aider confg

## Usage
1. Clone repository
```bash
git clone https://github.com/hypn4/my-aider-config
```

2. Copy configs to your project dir(such as vscode workspace)
```bash
cp my-aider-config TO_YOUR_PROJECT_DIR

3. Run `aider`
```bash
aider

# Or pass arguments
aider -c CONFIG_FILE_PATH
```

```
